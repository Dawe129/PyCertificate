# PyCertificate

Simple web quiz app for Python certification practice.

## What this project includes

- Python 1 quiz
- Python 2 quiz (questions 2-40)
- Immediate answer validation (correct/wrong)
- Multi-answer question support
- Optional question order randomization
- Final score summary after test completion
- Hidden correct answers with a per-question "Show answer" button

## How to run

1. Open `python1.html` in your browser.
2. Select a quiz from the menu.
3. Start normal mode or randomized mode.

No build tools or dependencies are required.

## Project structure

- `python1.html` - main app (UI, question data, quiz logic)
